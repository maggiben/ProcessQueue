# Security Policy

## Supported Versions

Security fixes are applied to the latest published version.

## Reporting a Vulnerability

Please report security issues privately by emailing:

- security@process-queue.dev

Include as much detail as possible:

- Affected versions
- Reproduction steps or proof of concept
- Potential impact

You should receive an acknowledgment within 3 business days. We will work on a
fix and coordinate disclosure once a patch is available.
